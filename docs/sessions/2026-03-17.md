### Troy's Core OSRS Insights (MEMORIZED)

**1. The Core Mechanic:**
- Everything uses **mouse clicking** (left-click or right-click)
- Left-click: default action
- Right-click: context menu with available actions
- NO KEYBOARD MOVEMENT EVER

**2. Three Progression Paths (Player Choice):**
- Quest path (story-driven)
- Skilling path (PvE leveling, gathering, crafting)
- Combat/PvP path (fighting, dungeoneering, PvP zones)
- All feed into shared economy

**3. Economy Design:**
- GE is optional (Ironman mode has no GE)
- Players CAN ignore GE and still have fun
- **Free market must always be viable** (flipping, trading)
- All paths contribute loot: PvE drops, skilling resources, PvP drops

**4. Combat System (Simple & Grindy):**
- Click enemy, select attack style from menu
- Attack style determines skill that levels: Stab=Attack, Slash/Crush=Strength, Block=Defence
- Different weapon types have different attack styles
- **Higher skill = higher-tier equipment access (hard gate)**
- Gear progression: level skill → unlock better equipment → kill harder enemies → progress faster
- Simple to play, grindy to master

**5. Skill Progression (THE GRIND IS ESSENTIAL):**
- **Max level: 99 for every skill**
- **XP curve is exponential: Level 92 = 50% of XP to 99**
- This means: Levels 1-92 = same XP as 92-99
- Early levels (1-30): Quick, rewarding
- Mid levels (30-60): Noticeable grind
- Late levels (60-85): Real grinding required
- Endgame (85-99): Extremely grindy, takes months of dedication
- Higher levels take exponentially longer

**6. What Keeps Players Engaged:**
- The grind itself is satisfying
- Working toward next best equipment tier
- Hard work to rank up at later levels
- Milestone progression (every level is earned)
- Freedom to choose what to grind
- Prestige of 99 (but not mandatory)

**System Design Insight - Interconnected Economy:**
- Each skill has TIERS based on level (logs → oak logs → yew logs)
- Higher tier = higher XP + better output
- All tiers feed economy: Woodcutting logs → Fletching → bows/arrows → players use/buy
- Fishing → Cooking (fish → food), Combat → Prayer (bones from kills)
- Everything connected: progression in one skill creates demand for another
- This is THE core OSRS design principle

**Post-Launch (not MVP):**
- Ranged weapon attack styles (only Magic + basic melee in MVP)
- Advanced skills (Slayer, Agility, Thieving, etc.)
- Special attacks
- PvP zones
- Multiple biomes

**Q2.2: Concurrent players & architecture decision?**

**Q3.3: Loot Systems & Economy (LOCKED IN)**
- **Research Complete:** Exact OSRS drop tables + economy interconnection
- **Document Created:** `LOOT_SYSTEM.md` (12,242 lines)
- **Key Insight (ULTRATHINK):** RAW MEAT is non-negotiable, it's the FEATURE not a limitation
  - Raw meat drops FORCE Cooking skill dependency
  - Without raw meat: Cooking becomes optional → food becomes abundant → economy breaks
  - With raw meat: Cooking is REQUIRED → creates natural interconnection
  - **The constraint IS the feature**
- **Drop Table Structure (Exact OSRS):**
  - Chicken: Always → Bones + Raw Chicken + Feathers (5-15)
  - Cow: Always → Bones + Raw Beef + Cowhide
  - Goblin: Always → Bones + Coins
  - Spider: Always → Bones (rarely other drops)
  - Giants: Always → Bones + Coins (50-100)
- **Raw vs Cooked (CRITICAL):**
  - Drops = RAW ONLY
  - Raw meat cannot be eaten (healing = 0)
  - Must cook → Transforms to cooked → Now healable
  - Cooked food heals; Raw meat doesn't
- **Economy Loop:**
  - Combat → Raw Meat drops
  - Cooking → Transforms meat to food
  - Combat player → Uses cooked food for healing
  - Fishing → Raw fish drops
  - Cooking → Transforms fish to food
  - Creates organic demand/supply
- **Money Sinks:** Food consumption, Gear upgrades, Prayer potions
- **Money Sources:** Coin drops, Resource sales (excess to GE)

  - Drives bank trips, creates resource scarcity, enables economy
  - Remove 28 slots = economy breaks

- **Non-stackable items:** Weapons, armor, tools (1 item = 1 slot)
- **Looting Bag (Wilderness):**
  - Drops from any Wilderness mob

  - Holds 28 extra items (only in Wilderness)
  - Auto-deposits tradeable loot if "Open"
  - Cannot be banked, only withdraw at bank
  - Reward for high-risk Wilderness grinding
- **Weight System:** Every item has weight in kg
  - Total weight affects run energy drain
  - Negative weight items (graceful gear) = optimization
  - Strategic loadout choices (heavy armor vs. light gear)
- **Drop Mechanics:**
  - Right-click "Drop" removes item from inventory
  - Appears on ground, visible to all
  - Own items picked up instantly
  - Can carry multiple items via stacking
- **Bank System (MVP):**
  - Unlimited slots (large capacity 1000+)
  - Deposit/withdraw items
  - Stackable items merge
  - NPCs have "Bank" option
  - No permanent inventory expansion from bank
