## DatabaseManager.java (Implemented)

**Location:** `server/src/main/java/com/osrs/server/database/DatabaseManager.java`

**Key Methods:**
- `initialize()` — Initialize pool and return singleton instance
- `getConnection()` — Get connection from pool
- `isHealthy()` — Health check (test connection)
- `getPoolStats()` — Pool statistics (active/idle/pending)
- `shutdown()` — Graceful shutdown

**Configuration:**
```java
JDBC_URL = "jdbc:sqlserver://localhost:1433;databaseName=osrsmmorp;encrypt=false;trustServerCertificate=true;"
DB_USER = "sa"
DB_PASSWORD = ""  // Windows auth
MAX_POOL_SIZE = 10
MIN_IDLE = 2
CONNECTION_TIMEOUT_MS = 5000
IDLE_TIMEOUT_MS = 600000
MAX_LIFETIME_MS = 1800000
```

**Usage in Server.java:**
```java
// Stage 2: Initialize database
database = DatabaseManager.initialize();
LOG.info("✓ Database initialized");

// ... server runs ...

// Shutdown
if (database != null) {
    database.shutdown();
}
```
